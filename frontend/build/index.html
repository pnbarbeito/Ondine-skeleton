Ondine Skeleton

Bienvenido a Ondine

Un framework PHP ligero para construir APIs RESTful con autenticación JWT y sistema de perfiles.

Crear Controllers Nuevos

Para agregar nuevos endpoints, crea una clase en src/Controllers/ que herede de BaseController. Define métodos como get(), post(), etc., y registra el controller en src/Router.php.

Ejemplo:

class MiController extends BaseController {
    public function get() {
        return $this->json(['mensaje' => 'Hola Mundo']);
    }
}

Registra en Router: $router->addRoute('GET', '/mi-endpoint', [MiController::class, 'get']);

Sistema de Perfiles y Permisos

Ondine utiliza un sistema de perfiles con permisos definidos en JSON. Cada perfil tiene permisos para endpoints específicos:

  • admin: Permisos completos (valor 1).
  • profiles: Gestionar perfiles (0: solo lectura, 1: lectura/escritura).
  • users: Gestionar usuarios (0: solo lectura, 1: lectura/escritura).

Los permisos se configuran en la base de datos y se validan en los middlewares. Para nuevos endpoints, agrega los permisos correspondientes en el perfil.
